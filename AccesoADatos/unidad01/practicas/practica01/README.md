## Comandos básicos de Composer

Para instalar las dependencias del proyecto, ejecuta:

```bash
composer install
```

Para actualizar las dependencias a las versiones más recientes permitidas por tu archivo `composer.json`, ejecuta:

```bash
composer update
```
